List Interface

Definition:
A List in Java is an ordered collection (also called a sequence). It allows duplicate elements and preserves insertion order. You can access elements by their index (position).

Simple Understanding

Think of a shopping list. You can write "Milk" twice (duplicates allowed), and the order in which you write items is preserved. Similarly, Java List keeps elements in the same order you added them.

Real-Life Example

Imagine a queue number list in a hospital. If you join at position 5, you’ll always remain 5th in line. If another patient joins with the same name, it’s still recorded (duplicates allowed).

Example

  // Example of List in Java
  import java.util.*;
  
  public class ListExample {
      public static void main(String[] args) {
          List shoppingList = new ArrayList<>();
  
          // Adding items
          shoppingList.add("Milk");
          shoppingList.add("Bread");
          shoppingList.add("Butter");
          shoppingList.add("Milk"); // duplicate allowed
  
          // Accessing by index
          System.out.println("First item: " + shoppingList.get(0));
  
          // Iterating
          for(String item : shoppingList) {
              System.out.println(item);
          }
      }
  }

Advantages & Disadvantages

  • Advantages:
    • Maintains insertion order.
    • Allows duplicates.
    • Index-based access.
  • Disadvantages:
    • Searching/removing in the middle can be slow (depends on implementation).

Specific Keywords & Concepts

  • ArrayList: Fast random access, but slow for inserts/removals in the middle.
  • LinkedList: Better for frequent insertions/removals, slower random access.
  • Vector: Legacy, thread-safe but less used now.
  • Stack: Legacy subclass of Vector, represents LIFO.

Notes

Use ArrayList when you mostly need fast random access. Use LinkedList when you insert/remove elements often in the middle.

Latest Java Updates

  • Java 9 introduced List.of(...) for creating immutable lists.
  • Java 16+ allows pattern matching with instanceof for safe type checks.
  • Streams API works directly with Lists for functional programming style.

Tips

For thread-safe List in modern Java, prefer CopyOnWriteArrayList over Vector. Also, always program to interface (List) instead of implementation.
